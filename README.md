# End-to-End Data Engineering Project

## Overview

This project demonstrates how to build a production-style Data Engineering pipeline from scratch.

The project covers the complete lifecycle of data engineering:

- Data Modeling
- SQL Server Database Design
- Python ETL
- Data Validation
- Data Warehouse
- Apache Airflow
- Docker
- Power BI
- GitHub
